					Lista de Listas( ha )

1. 	Escrever um programa que crie uma lista estática
	encadeada utilizando uma matriz de no máximo 10
	posições, o programa deverá permitir as operações de
	inserir e excluir elementos além de exibir a lista. A
	matriz deverá possuir 10 linhas e 2 colunas. A primeira
	coluna armazenará o valor informado pelo usuário e a
	segunda coluna a posição do próximo elemento. O
	usuário deverá informar a posição que deseja (incluir/
	excluir) cada elemento. Todos os testes de existência ou
	não do elemento deverão ser efetuados, seja na
	operação de inserção quanto na remoção.
	
2. 	Faça um programa que crie uma lista dinâmica
	encadeada. O usuário deverá informar quantos termos
	deseja inserir.

3. Faça um programa em que o usuário informe o valor
	que deseja remover de uma lista dinâmica encadeada.

4. Faça um programa em que o usuário informe a posição
	que deseja remover de uma lista dinâmica encadeada.
	
5. Faça um programa para gerenciar uma lista dinâmica
	encadeada. O usuário terá a opção de inserir quantos
	nós(elementos) desejar sequencialmente. O programa 
	também deverá permitir ao usuário eliminar nós tanto 
	pelo valor digitado quanto pela posição do nó. O pro-
	grama também deverá inserir nós na posição que o 
	usuário desejar.
	
6. Faça um programa para manipular lista dinâmica
	encadeada ORDENADA. O usuário terá a opção de
	inserir e/ou remover elementos pela posição ou pelo
	valor.

7. Faça um programa para manipular uma lista dinâmica
	encadeada CIRCULAR. Oferecer ao usuário as opções
	de inserir e remover elementos no modelo dos
	exercícios anteriores.

8. Faça um programa para manipular uma lista dinâmica
	DUPLAMENTE encadeada CIRCULAR. Oferecer ao
	usuário as opções de inserir e remover elementos no
	modelo dos exercícios anteriores.
