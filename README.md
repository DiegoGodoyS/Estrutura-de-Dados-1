# O que é?
Este repositório foi criado para servir de uma "base de códigos" para aprendizagem e compartilhamento de conhecimento da turma de ED1. Aqui estão códigos e suas respectivas listas de exercícios para praticar os conceitos. Para que sirva sendo uma boa base, quanto mais códigos melhor!!

# Porquê usar?
Ler códigos ajuda na aprendizagem em programação, seja dando uma noção inicial do que fazer ou como fazer soluções mais eficientes ou até mesmo mais legais.

# Como Usar?
> CONTRIBUA! -- Suba seus códigos para que possuamos as mais variadas soluções.
> 
> OBSERVE E APRENDA! -- Veja soluções diferentes para os mesmos problemas, tendo novas idéias para os seus e intendendo aquela parte da matéria que estava chatinha ;)

# Estrutura
O repositório está dividido nas seguintes pastas :
>EStrutura de dados 1:
* Alocação Dinâmica
* Lista
* Pilha
* $ Desafios

Cada pasta contém o conteudo com a qual foi entitulada, contudo, a pasta $ Desafios serve para galera que curte testar seus conhecimentos ou forçar mais um pouco.

# Nomeando os arquivos
Para facilitar a vida do pessoal, foi pensando o seguinte padrão simples para nomear os arquivos :
-   Numero do exercício
-   Nome de quem enviou( apenas para diferenciar )
- Ex : 01_AAATAB.c

Padrão definido até agora, se algo melhor for pensado este aquivo será modificao.
